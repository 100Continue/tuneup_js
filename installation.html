There are two ways you can setup tuneupjs in your project: you can install it manually, or you can use CocoaPods.

CocoaPods

Setting up tuneup using CocoaPods is the preferred way to install tuneup.

Step 1

In your Podfile add a line for "tuneup_js" like this: {% highlight ruby %} platform :ios pod 'tuneup_js' # other pods go here {% endhighlight %}

Step 2

Update your installed pods by running pod install. This will create a sub-folder named tuneup_js in your Pods directory.
