Contribute

Tuneup is an open-source project that accepts contributions from the community. If you would like to contribute, fork the project on GitHub and submit a pull-request.

Please make sure you follow the existing indentation conventions (two spaces, no tabs) and that you provide in-line documetation of how your new features work. Also note that the test-runner has been deprecated and no new patches will be excepted for it.
